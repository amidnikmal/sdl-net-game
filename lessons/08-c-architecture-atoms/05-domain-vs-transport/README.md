# 05 Domain Vs Transport

## Смысл атома

Domain-объект и сетевой пакет — это не одно и то же.

Правильнее:
- внутри логики держать осмысленный `struct`;
- на границе сети кодировать его в буфер байтов;
- на приёме декодировать обратно.

## Что сделать

Доделай `encode_position_packet()` и `decode_position_packet()`.

Требуемый вывод:

```text
hex: 03 01 2c 02 bc
decoded: 3 300 700
```
