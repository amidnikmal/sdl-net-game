# 06 No Globals Context

## Смысл атома

Глобалы удобны в начале, но быстро делают непонятным:
- кто меняет состояние;
- откуда пришло значение;
- какой порядок жизни у данных.

Контекст-структура и явная передача указателя обычно делают систему чище.

## Что сделать

Доделай `handle_command()` и `run_frame()`, чтобы всё шло через `struct app_context *ctx`.

Требуемый вывод:

```text
ticks: 3
```
