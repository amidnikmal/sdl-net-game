# 02 Opaque Handle

## Смысл атома

Если всем видна внутренняя структура модуля, любой код может сломать инварианты напрямую.

Opaque type нужен, чтобы:
- скрыть внутренности;
- оставить только API;
- разрешить менять реализацию без переписывания вызывающего кода.

## Что сделать

Доделай реализацию счётчика, не раскрывая его поля в заголовке.

Требуемый вывод:

```text
value: 3
```
