# 04 Single Source Of Truth

## Смысл атома

Если одна и та же сущность хранится в двух полях и их можно менять независимо, они почти гарантированно разъедутся.

Лучше:
- одно каноническое состояние;
- или один централизованный путь обновления, который синхронизирует все зависимые поля.

## Что сделать

Доделай `entity_set_position()` и `entity_move()`.

Требуемый вывод:

```text
world: 5 7
screen: 5 7
```
