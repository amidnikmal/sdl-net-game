# 01 Ownership Contract

## Смысл атома

Главный вопрос:

```text
Кто создаёт объект?
Кто им владеет?
Кто обязан его освободить?
```

Если на это нет ясного ответа, в C быстро приходят:
- утечки
- double free
- invalid free

## Мини-правило

- creator обычно создаёт объект;
- destroyer обычно его освобождает;
- функция вида `send_*()` или `print_*()` обычно **не** забирает ownership.

## Что сделать

Доделай `packet_create()` и `packet_destroy()`.

Требуемый вывод:

```text
text: ping
size: 4
```
