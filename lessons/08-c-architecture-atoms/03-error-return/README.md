# 03 Error Return

## Смысл атома

Helper-функция не должна сама завершать весь процесс через `exit()`, если это не её прямой уровень ответственности.

Правильнее:
- вернуть статус;
- дать вызывающему коду решить, что делать дальше.

## Что сделать

Доделай `parse_port()`, чтобы она:
- не падала на плохом вводе;
- возвращала статус;
- заполняла `out_port` только при успехе.

Требуемый вывод:

```text
ok: 8080
err: invalid
```
