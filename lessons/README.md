# Lessons

Здесь лежат маленькие практические уроки по мотивам `sdl-net-game`.

Текущий принцип:
- один маленький законченный урок лучше, чем 7 недоделанных;
- урок = `README + template.c + solution/ + test.sh`;
- `test.sh` здесь не про unit tests, а про критерий: скомпилировалось, запустилось, не упало.

Сейчас полноценно сделаны:
- `01-gcc-compilation`
- `02-makefile-basics`
- `03-sockets-udp`
- `04-packet-serialization`
- `05-sdl-graphics`
- `06-client-server-protocol`
- `07-game-loop`

Курс закрыт минимальным набором из 7 маленьких уроков. Дальше лучше не раздувать систему, а реально проходить и использовать её как справочник.

Дополнительный трек:
- `08-c-architecture-atoms` — маленькие архитектурные best practices на C, с которыми можно играться отдельно от основной линейки.
- `09-guided-project-fixes` — пошаговый маршрут по реальным проблемам этого репозитория, чтобы чинить проект через обучение, а не вместо него.
