# 02 Remove Small Heap Copies

## Проблема

В проекте есть маленькие промежуточные `malloc()` ради временных копий:
- header
- body

Это создаёт лишние утечки и запутывает поток данных.

## Где смотреть

- `src/serv_network.c`
- `src/cl_network.c`

## Сначала пройти

- `lessons/04-packet-serialization`
- `lessons/08-c-architecture-atoms/01-ownership-contract`

## Что нужно сделать

Убрать ненужные heap-копии там, где достаточно:
- локального буфера;
- указателя со смещением;
- прямого чтения из входного пакета.

## Чего НЕ делать

- Не переписывать весь парсер заново.
- Не смешивать это с исправлением ownership builder-функций.
- Не менять поведение протокола.

## Минимальный критерий готовности

В `serv_parse_packet()` и `cl_read_packets()` больше нет бессмысленного `malloc()` только ради временного header/body.

Ты можешь объяснить:
- почему для fixed-size header не нужен heap;
- где лучше использовать pointer arithmetic вместо лишней копии.
