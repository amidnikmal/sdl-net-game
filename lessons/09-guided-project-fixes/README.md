# 09 Guided Project Fixes

## Что это

Это не “ещё один курс”.

Это мост между:
- учебными уроками `01..07`
- архитектурными атомами `08`
- реальным кодом `src/`

Задача трека:
- не чинить проект за тебя напрямую;
- а дать тебе очень маленькие, проходимые ступеньки;
- чтобы ты сам дошёл до реальных фиксов в `sdl-net-game`.

## Главный принцип

Теория должна быть отдельно.
Конкретный баг — отдельно.
Связь между ними — явная.

Поэтому каждая задача здесь содержит:
- **Проблему в проекте**
- **Где смотреть**
- **Что нужно сделать**
- **Что НЕ делать**
- **Критерий готовности**
- **Какие уроки/атомы пройти перед задачей**

## Как проходить

Для каждой задачи:

1. Сначала открыть указанные `lesson` / `atom`.
2. Потом открыть целевые файлы проекта.
3. Сделать маленький фикс.
4. Проверить только локальный критерий задачи.
5. Остановиться.

Не надо:
- параллельно чинить 5 вещей;
- переписывать архитектуру целиком;
- “заодно” делать красивый рефакторинг.
